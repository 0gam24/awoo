전 국민 대상 정부 지원금 가이드 사이트. 페르소나 6개 + 지원금 10개(시작) 데이터 모델로, 이슈 중심 헤드라인 + 페르소나 매칭 + 소득 자가진단 인터랙티브 체커.

- **도메인**: awoo.or.kr (.or.kr = 한국 비영리단체 전용)
- **운영 형태**: 비영리 정보 안내 사이트 (신청 대행 없음, 공식 신청처로 외부 연결만)
- **컨셉 출처**: 01_지원금가이드/project/ (Claude Design 핸드오프 번들)
- **콘텐츠**: 6 페르소나 (사회초년생·자영업·신혼육아·중장년·저소득·농업) × 10 지원금 (월세지원·신혼특공·청년도약계좌·내일배움카드·소상공인자금·예비창업패키지·부모급여·신중년고용장려금·기초생활보장·청년후계농)

**Why:** 사용자가 비영리법인 도메인을 명시했고 프로토타입 README도 "정보 안내, 신청 대행 없음"을 강조함. 이 정체성이 E-E-A-T 신호와 GEO 인용 친화도를 결정함.

**How to apply:** Organization 스키마는 NGO 또는 NonprofitOrganization 사용. 푸터·About 페이지에서 비영리 + 신청 대행 없음 명시. 모든 콘텐츠는 외부 권위(정부 부처) 1차 자료 인용으로 작성.
