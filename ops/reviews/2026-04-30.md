## 6. 회귀 위험 / 주의

- **P0-1 (llms-full.txt)**: 파일 크기 폭증 가능 → issues 누적량 검증, 100KB 초과 시 페이지네이션 검토. 본 사이클은 일단 단일 파일.
- **P0-2 (Org/WebSite JSON-LD)**: 기존 `BreadcrumbList` 자체 출력 페이지와 중복 가능 → BaseLayout `breadcrumbs` prop 호출 페이지에서 인라인 schema 직출력 제거.
- **P0-4 (모든 hub BreadcrumbList)**: review-regression의 P0-7과 충돌 가능 (`issues/topics/[term].astro` 인라인 + BaseLayout prop 이중 출력) → P0-7 작업 시 일원화로 동시 해결.
- **P0-7 (cat-* 토큰)**: 라이트모드 색상 변화 없음 보장 후 적용. 다크모드만 자동 동기 회복.
- **P0-8 (size guard)**: 임계값 50KB는 현재 홈 ~37KB 기준 안전 마진. 단 빌드 fail 발동 시 정확한 size 정보 출력 필수.
